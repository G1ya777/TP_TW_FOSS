
# Prerequisites
* having mongoDB installed and listening on the default port (27017)

# HOW TO USE

* open a console
* cd into p1 folder
* run  `npm i` to install node modules
* run `npx nodemon app.js` to start the express server
* open another console
* cd into p2 folder
* run `npm i` to install node modules
* run `npm start`
